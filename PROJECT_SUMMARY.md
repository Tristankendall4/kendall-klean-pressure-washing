## 🚀 Quick Start

### 1. Local Setup (5 minutes)
```bash
cd kendall-klean-pressure-washing
npm install
cp .env.example .env.local
# Edit .env.local with your business info
npm run dev
# Visit http://localhost:3000
```

### 2. Deploy to Vercel (5 minutes)
```bash
git add .
git commit -m "Initial Kendall Klean site"
git push origin main
# Visit vercel.com → Import GitHub repo → Deploy
```

### 3. Add Images (30 minutes)
- Export before/after photos from existing site
- Optimize with TinyPNG
- Add to `/public/images/`
- Update image URLs in components

### 4. Launch (immediate)
- Add custom domain
- Update Google Business Profile
- Submit to Google Search Console
- Monitor analytics
