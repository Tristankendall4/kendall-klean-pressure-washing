## 🚀 Deployment Steps (One-Time Setup)

### 1. Push to GitHub
```bash
cd /workspaces/kendall-klean-pressure-washing
git add .
git commit -m "Production ready: Real images, logo, phone, social media"
git push origin main
```

### 2. Deploy to Vercel
1. Go to [vercel.com](https://vercel.com)
2. Click "New Project"
3. Select your GitHub repository: `Tristankendall4/kendall-klean-pressure-washing`
4. Click "Import"
5. Vercel auto-detects Next.js - click "Deploy"
6. Wait 2-5 minutes for deployment to complete
7. You'll get a live URL (e.g., `kendall-klean-pressure-washing.vercel.app`)

### 3. Configure Custom Domain
1. In Vercel dashboard, go to your project
2. Settings → Domains
3. Add `kendallklean.com`
4. Choose your domain registrar (GoDaddy, Namecheap, etc.)
5. Update nameservers to Vercel's nameservers
6. Wait 24-48 hours for DNS propagation

### 4. Set Up Google Services
- **Google Business Profile**: https://business.google.com/
  - Verify business
  - Add photos (your before/afters)
  - Add service areas
  - Link to website

- **Google Search Console**: https://search.google.com/search-console/
  - Add property for kendallklean.com
  - Verify with DNS
  - Submit sitemap: https://kendallklean.com/sitemap.xml
  - Monitor indexing

## 🔧 Troubleshooting

### Phone number shows old value
- Check `.env.local` has `NEXT_PUBLIC_PHONE_NUMBER="+1-309-613-6609"`
- Rebuild if needed: `npm run build`

### Social media links not working
- Links are in Footer component
- Environment variables in `.env.local`
- Should work immediately after deployment

### Images not showing
- Check files exist in `/public/images/`
- Browser might need cache clear: Ctrl+Shift+Delete
- Check Network tab for 404 errors

### Build fails
- Run `npm install` again
- Clear `.next` folder: `rm -rf .next`
- Run `npm run build` again
